# 《DageChat 全景评估报告》
**——下一代去中心化、抗审查、反追踪的硬核通讯堡垒**

## 摘要 (Executive Summary)
**DageChat** 是一款基于 Nostr 协议构建的“全场景加密通讯” Web 端软件。它摒弃了传统中心化服务器架构，以“密钥即身份”为绝对核心。

通过对源码的深度拆解，DageChat 展现出了远超同类产品的极客精神：它不仅融合了 **AES-GCM 本地加密、NIP-59 双重盲化加密、PoW 防御机制**，更在底层首创了**“共享密钥幽灵群组”**、**“图像重绘防隐写”**以及**“跨语言底层密码学修正”**。它在彻底消灭网络元数据追踪的同时，实现了媲美微信/Telegram 级别的流畅体验。

## 一、 核心杀手锏：彻底消灭网络追踪与安全隐患

DageChat 在安全与隐私保护上，拥有以下其他主流聊天软件（甚至去中心化竞品）所不具备的硬核防御细节：

### 1. 彻底消灭网络元数据追踪 (NIP-59 Gift Wrap)
*   **传统痛点：** 传统加密软件（如 Signal 早期或普通 Nostr）即使内容加密，监听者依然知道“公钥A在晚上8点给公钥B发了消息”，从而轻易绘制出用户的社交关系网。
*   **DageChat 方案：** 默认强制开启 NIP-59 协议。你的消息不仅被加密，系统还会自动生成一个**一次性临时密钥（Ephemeral Key）**，将整条消息（包括真实的发送者、接收者、时间戳）像“俄罗斯套娃”一样再次打包盲化。
*   **结果：** 审查者在公网中只能看到“一堆毫无规律的随机乱码在传递”，完全无法得知是谁发给谁的，实现了真正的**网络隐身**。

### 2. 首创“幽灵群聊”架构 (Ghost Groups ⚡)
*   **传统痛点：** 去中心化网络中的群聊发言，发送者 ID 都是公开绑定的。
*   **DageChat 方案：** 创建群组时支持生成“共享密钥群”。群内所有人共享同一对私钥进行 NIP-59 收发。发言时强制截断真实身份，仅附带随机 Nonce 和自选化名 (Alias)。
*   **结果：** 实现了**终极的群聊匿名性**。在区块链和底层网络看来，整个群自始至终只有“一个账号”在自言自语，从密码学上彻底切断了溯源群内具体发言者的可能。同时，DageChat 也支持基于 **PoW（工作量证明）**的普通端到端加密群，以算力拦截恶意刷屏。

### 3. Canvas 像素级图像防毒与防隐写引擎 (独创)
*   **传统痛点：** 攻击者经常在图片的 Alpha（透明）通道隐藏 Shellcode，或通过 EXIF 数据暴露用户的精确 GPS 定位与设备型号。
*   **DageChat 方案：** 源码内置硬核的本地重绘引擎（`processImageFile`）。用户上传任何图片，前端会隐式创建一个 Canvas，**强制使用纯白色 `#FFFFFF` 铺底绘制，彻底剥离所有透明通道和隐藏元数据**，并强制转换为纯净的 JPEG 格式。
*   **结果：** 在客户端物理层面直接摧毁了图片隐写术和恶意代码传播的路径，实现了“所见即所得”的绝对干净。

### 4. 底层密码学劫持修正：100% 跨平台兼容
*   **传统痛点：** Nostr 工具库（nostr-tools）在处理 NIP-44 v2 加密时，由于椭圆曲线 ECDH 推导的 Y 坐标奇偶性黑盒处理，常导致 Web 端与 Python/Go 等后端语言加密不互通。
*   **DageChat 方案：** 极其硬核地通过拦截 `Uint8Array.prototype.subarray`，强行劫持并捕获原始的完整共享密钥（Shared Secret），然后再通过原生 WebCrypto API 的 HKDF 算法重新派生加密轮钥。
*   **结果：** 完美解决了跨语言环境下的加密错位问题，保证了与任何 Python/Go 节点、机器人的 100% 加解密兼容。

### 5. 军事级本地金库 (Local Vault) 与防窥机制
*   **本地全盘加密：** 用户的聊天记录、通讯录、个人资料缓存到浏览器 `localStorage` 时，全部使用用户的公私钥进行 NIP-04 级别的二次加密。搜索功能完全基于**本地内存解密后**运行，不走任何服务端。
*   **防劫持锁屏：** 设定自动锁定时间（1-600分钟）。一旦超时或无操作，**内存中的明文私钥瞬间销毁**，WebSocket 连接切断。即便设备被物理缴获，拿到的也只是一堆无法破解的 AES-GCM 密文。
*   **系统通知防泄露：** 调用系统通知时，DageChat 强制剥离聊天内容，仅显示“收到一条新消息”，严防操作系统底层日志（如 iOS/Windows 通知中心）记录聊天明文。

## 三、 🚀 超越“玩具”：企业级的 IM 交互体验

很多去中心化软件虽然安全，但极为难用。DageChat 在纯前端（无任何后端数据库）的极限条件下，实现了媲美微信/Telegram 的丝滑交互体验：

1.  **无限嵌套历史记录转发：**
    支持“合并转发”和“逐条转发”。合并转发时，在前端将多条消息打包成一个 JSON 卡片。接收方点击卡片，会弹出**带堆栈深度的历史记录查看器**，甚至支持卡片内的多层再次嵌套（套娃解析），极大方便了取证与信息留存。
2.  **增量云端漫游 (Kind 3 合并算法)：**
    用户的通讯录、群组、黑名单，通过 Python 级 NIP-44 强加密后上传到去中心化节点。换设备登录时，支持一键“增量拉取合并”，不再出现去中心化工具常见的“换手机就变孤儿、丢好友”的尴尬。
3.  **多节点智能并发与防抖引擎：**
    同时并发连接 3 个以上的 Relay 节点。底层自带 `seenEventIds` 哈希去重。面对网络拥堵和群聊消息洪峰，内置 `batchUpdateTimer` 防抖机制，把 100 条瞬间到达的消息合并为 1 次 UI 渲染，保证 DOM 不卡顿，体验极其丝滑。
4.  **自由的数据主权：**
    提供全局记录的一键导出（生成精美的 HTML 或 TXT）。你的社交资产和聊天记录真正属于你自己，可以随时带着数据离开。
